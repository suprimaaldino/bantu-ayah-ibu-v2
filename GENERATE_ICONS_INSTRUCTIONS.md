# Cara Generate PWA Icons

## Langkah-langkah:

1. **Buka file `generate-icons.html`** di browser (double-click file)

2. **Download 3 icons** yang muncul:
   - `icon-192.png` (192x192)
   - `icon-512.png` (512x512)
   - `apple-touch-icon.png` (180x180)

3. **Pindahkan files** ke folder `public/`:
   ```
   public/
   ├── icon-192.png
   ├── icon-512.png
   └── apple-touch-icon.png
   ```

4. **Restart dev server** untuk melihat perubahan

## Atau gunakan online tool:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Upload gambar icon custom Anda dan download semua ukuran yang dibutuhkan.
