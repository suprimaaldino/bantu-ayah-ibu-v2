# PWA Icons

Untuk sementara, aplikasi menggunakan placeholder icons. 

## Cara Membuat Icon Custom:

1. **Buat gambar icon** dengan ukuran 512x512 pixels
   - Gunakan tools seperti Canva, Figma, atau Photoshop
   - Design harus simple dan recognizable
   - Gunakan warna brand: Purple (#8B5CF6) dan Pink (#EC4899)

2. **Generate berbagai ukuran** menggunakan:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator
   
3. **Replace files** di folder `public/`:
   - `icon-192.png` (192x192)
   - `icon-512.png` (512x512)
   - `apple-touch-icon.png` (180x180)
   - `favicon.ico`

## Temporary Icons
Saat ini menggunakan emoji-based icons sebagai placeholder.
